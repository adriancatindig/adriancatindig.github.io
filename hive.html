Hive

  • Category: Web and Machine Learning Application
  • Technologies used: Django, Django-tenants, XGBoost Machine Learning algorithm, Stripe, Chart.js, PostgreSQL, Scikit-learn, Pandas, Numpy, Bootstrap, HTML, CSS, JavaScript
  • Source Code: https://github.com/adrn1201/hive

Description

Hive is a multi-tenant Software as a Service (SaaS) platform, offering an AI-powered Order Processing System that wholesale businesses can utilize by paying a subscription fee. This system enables wholesalers to manage their inventory, retailers, and orders. Additionally, it incorporates the XGBoost Machine Learning algorithm, which helps wholesalers predict the demand for each of their products. Retailers can also place their orders through Hive's e-commerce/order portal.
